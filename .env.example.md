# `.env.example` — configuration template

## Purpose

The template of environment variables the server reads at startup (`config.rs`). Copy it to `.env`
(or export the vars in your process manager) and fill in real values. Every setting has a default
except the ones noted below, so a minimal deployment only needs `DATABASE_URL` and `JWT_SECRET`.

## Variables

| Variable | Default | Meaning |
|----------|---------|---------|
| `PORT` | `3000` | HTTP listen port |
| `DATABASE_URL` | — | PostgreSQL DSN (required) |
| `JWT_SECRET` | — | HS256 signing secret for device tokens (**required**; use a long random value) |
| `TOKEN_TTL_DAYS` | `365` | device token lifetime |
| `CHANGES_RETENTION_DAYS` | `0` | prune relay journal rows delivered to every device after N days (`0` = keep) |
| `LINES_GC_DAYS` | `30` | reclaim line tombstones older than N days |
| `DB_MAX_CONNECTIONS` | `10` | pool size |
| `DB_ACQUIRE_TIMEOUT_SECS` | `10` | fail fast instead of hanging when the pool is exhausted |
| `DB_IDLE_TIMEOUT_SECS` | `600` | close idle connections |
| `DB_MAX_LIFETIME_SECS` | `1800` | recycle connections |
| `RATE_LIMIT_PER_MIN` | `0` | per-IP request budget (`0` = disabled; see gotcha) |
| `SHUTDOWN_GRACE_SECS` | `20` | graceful-shutdown deadline before forced exit |
| `LOG_JSON` | `false` | structured JSON logs (set `true` in production) |
| `RUST_LOG` | `info` | tracing filter |

## Notes & gotchas

- **`JWT_SECRET` must be a strong random secret and kept stable** — rotating it invalidates every
  device token (all devices must re-login). Never ship the placeholder value.
- **`RATE_LIMIT_PER_MIN` keys on the peer IP.** Behind a reverse proxy every request carries the
  proxy's IP, so a non-zero value would throttle all clients as one — leave it `0` and rate-limit at
  the proxy. Enable it only when the server faces clients directly.
- `CHANGES_RETENTION_DAYS=0` keeps the relay journal forever; set a value once you have many devices
  and want bounded growth (rows are only pruned after delivery to all of a user's devices).
